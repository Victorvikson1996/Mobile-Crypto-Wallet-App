## Install & Build 

First, Make sure you have  NPM and yarn installed 

Install: `yarn` or `yarn Install`

Run Project Locally:  `Yarn Dev or yarn start` 

For Windows PC Open two terminal Run: `Yarn Start` & ` yarn react-native run-android`
